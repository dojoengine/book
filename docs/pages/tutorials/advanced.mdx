# Advanced Dojo Tutorials

Welcome to the advanced Dojo tutorials!
These guides assume you have completed the [Dojo Starter](./dojo-starter) tutorial and are familiar with the basic concepts of Dojo development.

## Available Tutorials

### [Onchain Chess](./onchain-chess/README)

Build a fully functional chess game on Starknet using Dojo.
This tutorial covers advanced ECS patterns, complex game state management, and sophisticated user interactions.

**What you'll learn:**

- Advanced ECS modeling for complex game states
- Implementing game rules and validation in Cairo
- Managing turn-based gameplay
- Optimizing gas costs for complex games

### [Deploy to Mainnet](./deploy-to-mainnet/main)

Learn how to deploy your Dojo world to Starknet mainnet.
This tutorial covers production deployment considerations, RPC provider setup, and mainnet-specific configurations.

**What you'll learn:**

- Setting up RPC providers for mainnet deployment
- Configuring production environment variables
- Understanding mainnet deployment costs
- Production deployment best practices

### [Deploy using Slot](./deploy-using-slot/main)

Deploy your Dojo applications using Slot, a streamlined deployment platform.
This alternative deployment method simplifies the process while providing powerful debugging and monitoring tools.

**What you'll learn:**

- Using Slot for simplified deployment
- Debugging deployed worlds with integrated tools
- Managing deployments through a web interface
- Monitoring world performance and usage

### [Katana, Starkli, and Scarb Setup](./katana-starkli-scarb/main)

A comprehensive guide to setting up and using the Starknet development toolchain alongside Dojo.
This tutorial covers advanced local development workflows and toolchain integration.

**What you'll learn:**

- Advanced Katana configuration and usage
- Integrating Starkli for account and contract management
- Optimizing Scarb for Dojo development
- Setting up comprehensive development environments

## Prerequisites

Before starting these advanced tutorials, ensure you have:

- Completed the [Dojo Starter](./dojo-starter) tutorial
- Basic understanding of Cairo programming
- Familiarity with Dojo's ECS architecture
- A working Dojo development environment
