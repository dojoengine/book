# Advanced Tutorials

This section contains advanced tutorials that build upon the foundational concepts from the [Dojo starter tutorial](./dojo-starter).

## Chess Tutorial Series

The [onchain chess tutorial](./onchain-chess) provides a comprehensive walkthrough of building a complex, fully-featured game using Dojo.
This tutorial series covers:

- Setting up game entities and components
- Implementing game actions and logic
- Managing complex state transitions
- Writing comprehensive tests
