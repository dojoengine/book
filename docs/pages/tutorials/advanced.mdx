# Advanced Tutorials

Welcome to the advanced Dojo tutorials section.
These guides assume you have completed the [Getting Started](/getting-started) tutorials and are comfortable with basic Dojo concepts.

## Available Advanced Tutorials

### Onchain Chess
A comprehensive multi-part tutorial building a fully functional chess game with Dojo.
This tutorial demonstrates advanced patterns for game state management, move validation, and complex game logic.

- [Setup and Project Structure](/tutorials/onchain-chess/0-setup)
- [Implementing Game Actions](/tutorials/onchain-chess/1-action)
- [Move Validation and Game Logic](/tutorials/onchain-chess/2-move)
- [Testing Strategies](/tutorials/onchain-chess/3-test)

### Deployment Strategies
Learn different approaches to deploying your Dojo projects to production networks.

- [Deploy to Mainnet](/tutorials/deploy-to-mainnet/main) - Manual deployment with RPC configuration
- [Deploy using Slot](/tutorials/deploy-using-slot/main) - Managed deployment with Cartridge Slot

### Development Workflows
Advanced development patterns and tooling integration.

- [Katana, Starkli, and Scarb Integration](/tutorials/katana-starkli-scarb/main) - Working with the broader Cairo ecosystem

## Prerequisites

Before starting these advanced tutorials, ensure you have:

- Completed the [Dojo Starter](/tutorials/dojo-starter) tutorial
- Basic understanding of [Models](/framework/models), [Systems](/framework/systems), and [World](/framework/world)
- Familiarity with Cairo programming language
- Understanding of blockchain development concepts
