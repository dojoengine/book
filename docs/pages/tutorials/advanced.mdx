# Advanced Tutorials

Once you've mastered the basics of Dojo development, these advanced tutorials will help you build more sophisticated autonomous worlds and explore cutting-edge techniques.

## Complex Game Mechanics

### [Onchain Chess](./onchain-chess/README)

Build a fully-featured chess game on Starknet that demonstrates:

- Complex game state management
- Move validation and game rules
- Player interactions and turn-based gameplay
- Advanced testing strategies for game logic

This comprehensive tutorial walks through creating a complete chess implementation, from basic piece movement to advanced features like castling and en passant.

## Full-Stack Development

### [React Integration](./react)

Learn how to build modern web frontends for your Dojo games:

- Connect React applications to Dojo worlds
- Use the Dojo SDK for seamless blockchain integration
- Implement real-time game state updates
- Handle user wallet connections and transactions

### [Dojo React App](./dojo-react)

A complete example showing how to structure a production-ready Dojo application with React, including best practices for state management and user experience.

## Deployment and Infrastructure

### [Deploy to Mainnet](./deploy-to-mainnet/main)

Take your game to production with this guide covering:

- Mainnet deployment strategies
- Infrastructure setup and monitoring
- Security considerations for live games
- Performance optimization techniques

### [Deploy Using Slot](./deploy-using-slot/main)

Learn how to deploy your Dojo world using Slot, a deployment service that simplifies the process:

- Automated deployment workflows
- Infrastructure management
- Monitoring and maintenance

## Development Workflows

### [Katana, Starkli & Scarb](./katana-starkli-scarb/main)

Master the complete Starknet development toolkit:

- Advanced Katana configuration for local development
- Starkli for low-level blockchain interactions
- Scarb project management and optimization

## Prerequisites

Before diving into these advanced tutorials, make sure you have:

- Completed the [Dojo Starter](./dojo-starter) tutorial
- Basic understanding of Cairo and Starknet
- Familiarity with the Dojo toolchain
- Development environment properly configured

## Next Steps

These tutorials build upon each other and demonstrate different aspects of advanced Dojo development.
Start with the topic that best matches your current project needs, but consider working through all of them to gain a comprehensive understanding of the Dojo ecosystem.

For more specific guidance on particular aspects of Dojo development, check out the [Framework documentation](../framework) and [Toolchain reference](../toolchain).
