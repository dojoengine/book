# Advanced Tutorials

Ready to dive deeper into Dojo development?
These advanced tutorials will help you build more complex games and applications.

## Available Tutorials

### [On-chain Chess](/tutorials/onchain-chess/README)

Build a fully functional chess game on Starknet using Dojo.
This comprehensive tutorial series covers:

- Setting up the chess game structure
- Implementing game actions and moves
- Advanced testing patterns
- Complex game state management

Perfect for developers who want to understand how to build sophisticated turn-based games with Dojo.

## Prerequisites

Before starting these tutorials, make sure you're comfortable with:

- Basic Dojo concepts from the [starter tutorial](/tutorials/dojo-starter)
- Cairo programming fundamentals
- Entity Component System (ECS) patterns
