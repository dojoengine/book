# Advanced tutorials

Once you've mastered the basics of Dojo development, these advanced tutorials will help you explore more sophisticated patterns and techniques for building autonomous worlds.

## Production deployment

Learn how to deploy your Dojo projects to production networks:

- **[Deploy to mainnet](./deploy-to-mainnet/main)** - Complete guide for deploying your world to Starknet mainnet, including RPC provider setup, security considerations, and best practices
- **[Deploy using Slot](./deploy-using-slot/main)** - Streamlined deployment process using Slot's infrastructure

## Complex game mechanics

Build sophisticated on-chain games with advanced patterns:

- **[Onchain Chess](./onchain-chess/README)** - Complete implementation of chess as a Dojo game, covering complex game state management, move validation, and turn-based mechanics

## Integration tutorials

Connect Dojo with other technologies and frameworks:

- **[Katana, Starkli, and Scarb](./katana-starkli-scarb/main)** - Deep dive into the Starknet development toolchain and how it integrates with Dojo workflows

:::tip
Before diving into these advanced tutorials, make sure you're comfortable with the concepts covered in [Getting Started](../getting-started) and have completed at least one of the beginner tutorials like [Dojo 101](./dojo-starter) or [React Quickstart](./react).
:::
