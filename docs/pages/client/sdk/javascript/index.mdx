# Dojo.js

Dojo.js is the official JavaScript package for dojo. It provides a set of tools and utilities to interact with dojo and build JavaScript based apps.

### Prerequisites and Tooling

::::steps

### Install nvm

Easiest way to install nvm is to use the [nvm-install](https://github.com/nvm-sh/nvm#installing-and-updating) script:
Provides a flexible to use nodejs and switch between versions.

Once installed, you should install the latest LTS version of Node.js and use it as your default version:

```bash
nvm install --lts
nvm use --lts
```

### Install pnpm

```sh
npm install -g pnpm
```

Now run `pnpm --version` to check if it has properly been installed.
