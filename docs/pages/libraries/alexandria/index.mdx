## Alexandria

Alexandria is a community maintained standard library for Cairo 1.0.
It is a collection of useful algorithms and data structures implemented in Cairo.

<Callout type="warning">
  This library is built outside of the Dojo stack development.
  There is no online documentation yet but most of the implemented features are referenced in the readme files of crate.
</Callout>

### Overview

Its scope includes the following crates:

- [ASCII](https://github.com/keep-starknet-strange/alexandria/blob/main/packages/ascii/README.md)
- [Data Structures](https://github.com/keep-starknet-strange/alexandria/blob/main/packages/data_structures/README.md)
- [Encoding](https://github.com/keep-starknet-strange/alexandria/blob/main/packages/encoding/README.md)
- [Linalg](https://github.com/keep-starknet-strange/alexandria/blob/main/packages/linalg/README.md)
- [Math](https://github.com/keep-starknet-strange/alexandria/blob/main/packages/math/README.md)
- [Numeric](https://github.com/keep-starknet-strange/alexandria/blob/main/packages/numeric/README.md)
- [Searching](https://github.com/keep-starknet-strange/alexandria/blob/main/packages/searching/README.md)
- [Sorting](https://github.com/keep-starknet-strange/alexandria/blob/main/packages/sorting/README.md)
- [Storage](https://github.com/keep-starknet-strange/alexandria/blob/main/packages/storage/README.md)
- [Bytes](https://github.com/keep-starknet-strange/alexandria/blob/main/packages/bytes/README.md)

### Integration

Integrating Alexandria into your project is straightforward. Simply add the desired crates to your Scarb.toml file:

```toml
[dependencies]
alexandria_math = { git = "https://github.com/keep-starknet-strange/alexandria.git" }
alexandria_data_structures = { git = "https://github.com/keep-starknet-strange/alexandria.git" }
```
